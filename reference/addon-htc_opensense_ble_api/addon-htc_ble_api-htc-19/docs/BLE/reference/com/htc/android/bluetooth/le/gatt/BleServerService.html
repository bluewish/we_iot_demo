Class Overview

Represents a Bluetooth Low Energy service for the server role. An application should override this class to implement a service. A service derived from this class can be a primary or secondary service.

public void onCharacteristicRead (String address, int transId, int attrHandle, BleCharacteristic charObj)

Callback triggered when a remote client requests to read a local characteristic. By default, this function will return an included characteristic that has been added to this service using the addCharacteristic(BleCharacteristic) function. Alternatively, an application may choose to override this function to determine a value dynamically and send a response using the sendResponse(String, int, byte[], int) function manually.

Parameters
address Remote device address
transId Transaction Id for this operation
attrHandle Handle to the requested attribute
charObj Characteristic to be read
