Class Overview

Abstract base class, representing a Bluetooth LE Client Profile.

To implement a new low energy profile, an application must extend this class, initialize it with the desired profile ID and the necessary required or optional services. Included services

Included services are represented by overriding the BleClientService class.

Once this class has been extended for the desired profile, an application may call the connect(BluetoothDevice) or connectBackground(BluetoothDevice) functions to initiate an active or passive connection to the remote device.

public int connectBackground (BluetoothDevice device)

Prepares a background connection to a remote Bluetooth device.

A background connection allows the remote device to initiate a connection to this profile. After calling connectBackground(BluetoothDevice), the local device will periodically scan for incoming connections from the specified device. An application must call cancelBackgroundConnection(BluetoothDevice) to stop scanning.
